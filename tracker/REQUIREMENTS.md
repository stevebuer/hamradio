# Single Board Computer Tracker Project

This project configures a single board computer as an APRS tracker.

A second mode will allow the decoding of FT8 signals.

## Hardware

Yaesu FT-891 transceiver.

Pine Rock64 running Armbian (Debian 11 Bullseye)a

## Audio Interface

The audio from the radio will be fed into the audio jack of the pine64 board via a custom board with isolation transformer.
